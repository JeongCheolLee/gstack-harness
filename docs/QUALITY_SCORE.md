# 품질 점수 (Quality Score)

> 각 도메인과 아키텍처 레이어의 품질을 추적한다.
> 등급: A(우수) / B(양호) / C(기본) / D(미흡) / F(없음)

## 인프라 & 하네스

| 영역 | 등급 | 테스트 커버리지 | 비고 |
|------|------|----------------|------|
| 에이전트 설정 (AGENTS.md, CLAUDE.md) | B | N/A | 초기 세팅 완료 |
| 아키텍처 문서 | C | N/A | 제품 미정, 프레임워크만 |
| CI/CD | F | 0% | 미구성 |
| 린터/포맷터 | F | 0% | 미구성 |
| 테스트 하네스 | F | 0% | 미구성 |

## 제품 도메인

> 제품이 결정되면 여기에 도메인별 품질 점수를 추가한다.
